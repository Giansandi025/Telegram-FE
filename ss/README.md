CHAT - App is an application where every user can connect and chat with each other in real time.

## Getting Started

### Installation

- Clone This Front End Repo

```
git clone https://github.com/Giansandi025/Telegram-FE
```

- Go To Folder Repo

```
cd Telegram-FrondEnd
```

- Install Module

```
npm install
```
